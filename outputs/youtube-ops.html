Safe order

Do This In Order

  1. Run live status check.
  2. Run readiness summary for one clean go/no-go view.
  3. List upcoming broadcasts after the retry worker creates them.
  4. Run dry-run create-upcoming.
  5. Use batch retry if YouTube live activation is still locked.
  6. Create all upcoming as private.
  7. Review in YouTube Studio.
  8. Only then switch selected live to public/scheduled.

Common errors

What It Means

  1. `livePermissionBlocked`: YouTube 24-hour live activation wait is still blocking create.
  2. `liveStreamingNotEnabled`: live access is not active for this channel/account yet.
  3. `invalid_grant`: OAuth token needs to be recreated.
  4. `quotaExceeded`: wait for YouTube API quota reset.
After create succeeds

Studio Review

  1. Open YouTube Studio live control room.
  2. Confirm every stream is private.
  3. Check title includes "No Footage".
  4. Check description has no-footage disclaimer.
  5. Add thumbnail from Thumbnail Studio.
