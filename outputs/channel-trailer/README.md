# Channel Trailer

Files:

- `trailer.html`: animated 30-second visual trailer page
- `voiceover.txt`: script for voiceover
- `voiceover.aiff`: generated macOS voiceover audio if available

## How To Record

1. Open `trailer.html` in a browser.
2. Add it to OBS as a Browser Source or window capture.
3. Play `voiceover.aiff` as media/audio.
4. Record a 30-second MP4.
5. Upload it as the channel trailer.

## Safe Notes

- No match footage.
- No broadcast audio.
- No copyrighted music.
