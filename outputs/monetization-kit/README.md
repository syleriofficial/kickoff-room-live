# Monetization Kit

Generate sponsor and affiliate copy:

```bash
npm run monetize
```

Outputs:

- `monetization-kit.md`: copy/paste sponsor and affiliate text
- `monetization-kit.json`: machine-readable sponsor copy

Use this for:

- stream descriptions
- pinned chat sponsor slots
- live read scripts
- sponsor outreach
- early rate card

Do not promote betting unless legal/platform policy checks are complete.
